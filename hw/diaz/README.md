# Aplicación Web de Biblioteca

## Descripción

Sistema de gestión de biblioteca desarrollado como proyecto académico para el curso de Programación Web (NRC 27811). La aplicación permite administrar el catálogo de libros, usuarios y préstamos de manera eficiente.

## Características

## Requisitos

- Node.js (v14 o superior)
- npm

## Instalación

1. Clona el repositorio
    ```bash
    git clone https://github.com/Uriel8334/PRY_APP_WEB_BIBLIOTECA.git
    ```
2. Instala las dependencias:
    ```bash
    npm install
    ```
3. Configura las variables de entorno (si aplica)
4. Inicia la aplicación:
    ```bash
    npm run dev
    ```
