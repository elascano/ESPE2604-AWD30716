Product Manager

Manage your product inventory with ease. Add products, convert weights, and view your complete inventory in real-time.

Features

  • Add products with 9 input fields
  • Real-time weight conversion (kg, lb, gr)
  • Complete inventory table view
  • Search and filter by category
  • MongoDB Atlas database
  • Responsive Vue.js interface
  • Product details modal
  • Quick delete functionality
ℹ️ Getting Started:

1. Click "Add Product" to enter new products
2. Fill all required fields (marked with *)
3. Watch weight conversions update in real-time
4. Click "View Inventory" to see all products
5. Use search and filters to find products

Technology Stack

Java 11 JSP Vue.js 3 MongoDB Tomcat Maven
