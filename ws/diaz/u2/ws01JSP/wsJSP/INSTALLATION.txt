INSTALLATION REQUIREMENTS
=========================

Tu sistema necesita los siguientes componentes para ejecutar el proyecto:

1. JAVA DEVELOPMENT KIT (JDK) 11+
   =================================
   Descarga desde: https://www.oracle.com/java/technologies/downloads/
   O usa OpenJDK: https://adoptium.net/
   
   Instalación Windows:
   - Descarga el instalador .exe
   - Ejecuta y sigue las instrucciones
   - Verifica: java -version en Command Prompt
   - Agrega JAVA_HOME a variables de entorno del sistema

2. APACHE MAVEN 3.6+
   ==================
   Descarga desde: https://maven.apache.org/download.cgi
   
   Instalación Windows:
   - Descarga maven-3.x.x-bin.zip
   - Extrae a: C:\apache-maven-3.x.x (o donde prefieras)
   - Agrega a PATH: C:\apache-maven-3.x.x\bin
   - Verifica: mvn -version en Command Prompt
   - Restart Command Prompt después de cambiar variables de entorno

3. MONGODB ATLAS ACCOUNT
   ======================
   - Crea cuenta en: https://www.mongodb.com/cloud/atlas
   - Crea un cluster gratuito
   - Crea usuario de base de datos
   - Whitelist tu IP
   - Obtén connection string

4. ACTUALIZA CONFIGURACION
   ========================
   Archivo: src/main/resources/application.properties
   
   Reemplaza:
   mongodb.uri=mongodb+srv://USERNAME:PASSWORD@cluster.mongodb.net/products?retryWrites=true&w=majority

VERIFICACION DE INSTALACION
============================

Abre Command Prompt y ejecuta:

java -version
  Debe mostrar: openjdk version "11" o superior

mvn -version
  Debe mostrar: Apache Maven 3.6.0 o superior

PASOS PARA EJECUTAR
===================

1. Abre Command Prompt en el directorio del proyecto

2. Ejecuta:
   mvn clean tomcat7:run

3. Espera a que veas:
   [INFO] Webapp directory = C:\...\src\main\webapp
   [INFO] Webapp url: http://localhost:8080/product-manager

4. Abre navegador y accede:
   http://localhost:8080/product-manager/

5. Para detener: Presiona Ctrl+C en Command Prompt

TROUBLESHOOTING
===============

Si mvn command not found:
- Verifica que Maven esté en PATH
- Usa ruta completa: C:\apache-maven-3.x.x\bin\mvn clean tomcat7:run
- Reinicia Command Prompt

Si Java compilation fails:
- Verifica Java 11+ está instalado
- Verifica JAVA_HOME está configurado

Si connection to MongoDB fails:
- Verifica MongoDB Atlas cluster está corriendo
- Verifica connection string es correcta
- Verifica IP está en whitelist

ALTERNATIVA: USAR IDE
====================

Si prefieres no usar línea de comandos:

1. Descargar Eclipse IDE for Enterprise Java Developers
   https://www.eclipse.org/downloads/

2. O IntelliJ IDEA Community Edition
   https://www.jetbrains.com/idea/download/

3. Importar proyecto como Maven project
4. Right-click > Run As > Maven build
5. Goals: clean tomcat7:run

NOTES
=====
- Primera ejecución descarga dependencias (puede tomar 5-10 min)
- Aplicación accessible en http://localhost:8080/product-manager/
- Presiona Ctrl+C para detener el servidor Tomcat
